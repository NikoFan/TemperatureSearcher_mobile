# Temperature Searcher

>Проект для отслеживания температуры в городах

>В качестве API для получения погоды в городах: **openweathermap**
>>Данная API позволяет получить JSON файл с данными по погоде в различных городах мира

## Работа с API
> Этот процесс описан в файлах:
* `WeatherApiService.kt`
* `WeatherClient.kt`
> В начале работы приложения идет создание моделей по каждому городу. Города хранятся в файле `Cities.kt`
> В ViewModel идет получение к API и считывание JSON файла. После получения `JSON` строки из нее создается модель `Card`, которая сохраняется в `mutableStateListOf<Card>()`.

## Паттерн
> В проекте используется паттер `MVVM`. Выбор данного паттерна сделан для практики и тренировки.
